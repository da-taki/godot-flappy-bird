# Godot Flappy Bird

My first complete game made with Godot.
A small Flappy Bird clone.

## Controls

- Press Space/Enter to flap
- Avoid the pillars and screen borders
- Survive for as long as possible

## Built With

- Godot 4
- GDScript

## Assets

Visual assets are from the MegaCrash Flappy Bird Assets pack.

Sound effects are from Pixabay.

## What I Learned

This project helped me learn Godot scenes, nodes, signals, collisions, timers, animation, audio, autoloads, UI, procedural spawning, and scene transitions.
